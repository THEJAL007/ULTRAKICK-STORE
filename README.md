# ⚽ UltraKick Football Store

A modern, full-stack e-commerce web application for football gear.

## 🚀 Features
* **Dynamic Product Grid:** Fetches jerseys, boots, and balls from a MySQL database.
* **Shopping Cart:** Add/Remove items and calculate totals in real-time.
* **Responsive Design:** Optimized for both Desktop and Mobile users.
* **Checkout System:** Professional modal interface for order confirmation.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
* **Backend:** Node.js with Express
* **Database:** MySQL
* **Icons:** FontAwesome

## 📖 How to Run Locally
1. Clone this repository.
2. Run `npm install` to get the dependencies.
3. Set up your MySQL database using the provided schema.
4. Run `node server.js` to start the backend.
5. Open `index.html` in your browser.
